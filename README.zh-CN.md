## 项目简介

PetBuddy 是一个本地单机桌面宠物放置养成游戏。宠物会常驻在桌面上，用户可以通过右键菜单或状态面板进行互动，包括喂食、学习、工作、睡觉、购买道具和调整设置。

项目当前不包含账号系统、网络服务或数据库。数据保存在本地 JSON 存档中。

## 当前功能

- 透明、无边框、可拖动、可置顶的桌面宠物窗口
- 不同状态的宠物图片：空闲、开心、饥饿、学习、工作、睡觉
- 左键点击互动，可增加心情
- 右键菜单入口：喂食、学习、工作、睡觉、商店、状态、设置
- 学习任务：5 分钟完成，获得 1 学识
- 工作任务：根据学识解锁不同工作，完成后获得金币
- 睡觉任务：支持 5 / 15 / 40 分钟，完成后恢复心情
- 商店系统：普通食物、高级食物、玩具、小床升级
- 本地自动保存，重启后继续未完成任务的剩余时间
- 可调整宠物大小、置顶和状态文字显示
- 气泡提示、点击增加心情和点击开心动画默认启用，安静模式默认关闭且不在设置中提供修改

## 运行环境

- Python 3.11+
- PySide6
- Pillow
- PyInstaller，用于打包

创建 conda 环境并安装依赖：

```bash
conda create -n petbuddy python=3.11
conda run -n petbuddy python -m pip install -r requirements.txt
```

## 启动项目

在项目根目录使用 conda 环境运行：

```bash
conda activate petbuddy
python main.py
```

## 基础规则

| 内容 | 当前规则                               |
| --- |------------------------------------|
| 自然金币 | 基础每分钟 +1，受心情倍率影响                  |
| 心情下降 | 空闲每分钟 -0.5，学习/工作每分钟 -0.6，睡觉不下降     |
| 饱食度下降 | 空闲每分钟 -0.5，学习/工作每分钟 -0.6，睡觉每分钟 -0.2 |
| 点击互动 | 10 秒冷却，心情 +3                       |
| 学习 | 5 分钟，完成后学识 +1                      |
| 睡觉 | 5 / 15 / 40 分钟，心情分别 +5 / +18 / +50 |

### 心情倍率

| 心情 | 倍率 |
| --- | ---: |
| 80-100 | 1.5x |
| 50-79 | 1.0x |
| 20-49 | 0.7x |
| 0-19 | 0.3x |

## 存档

开发运行时，存档会写入项目根目录下的 `saves/` 文件夹。程序会维护 `save.json` 和 `save_backup.json`，保存时使用临时文件写入后替换正式存档，降低写入中断造成损坏的风险。
