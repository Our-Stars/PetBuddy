## 项目简介

PetBuddy 是一个本地单机桌面宠物放置养成游戏。宠物会常驻在桌面上，用户可以通过右键菜单或状态面板进行互动，包括喂食、学习、工作、睡觉、购买道具和调整设置。

项目当前不包含账号系统、网络服务或数据库。数据保存在本地 JSON 存档中。

## 当前功能

- 透明、无边框、可拖动、可置顶的桌面宠物窗口
- 不同状态的宠物图片：空闲、开心、饥饿、学习、工作、睡觉
- 左键点击互动，可增加心情
- 右键菜单入口：喂食、学习、工作、睡觉、玩耍、商店、状态、设置
- 喂食、学习、工作、睡觉和玩耍均通过弹出窗口选择具体操作后确认执行
- 喂食、玩耍、学习、工作、睡觉、商店、状态、设置和右键菜单使用统一温馨宠物主题配色（橙棕色系）
- 状态面板打开后会实时刷新当前任务剩余时间和 Buff 剩余时间
- 学习、工作或睡觉期间不能喂食、玩耍或点击互动
- 学习任务：支持 5 / 15 / 40 分钟，完成后分别获得 1 / 3.5 / 10 学识
- 工作任务：根据学识解锁不同工作，完成后获得金币
- 睡觉任务：支持 5 / 15 / 40 分钟，完成后恢复心情
- 商店系统：食物和玩具双页面，支持多种食物、玩具和固定时长 Buff
- 本地自动保存，重启后继续未完成任务的剩余时间
- 可调整宠物大小、置顶和状态文字显示
- 气泡提示、点击增加心情和点击开心动画默认启用，安静模式默认关闭且不在设置中提供修改

## 运行环境

- Python 3.11+
- PySide6
- Pillow
- PyInstaller，用于打包

创建 conda 环境并安装依赖：

```bash
conda create -n petbuddy python=3.11
conda run -n petbuddy python -m pip install -r requirements.txt
```

## 启动项目

在项目根目录使用 conda 环境运行：

```bash
conda activate petbuddy
python main.py
```

## 基础规则

| 内容 | 当前规则                               |
| --- |------------------------------------|
| 自然金币 | 基础每分钟 +1，受心情倍率影响                  |
| 心情下降 | 空闲每分钟 -0.5，学习/工作每分钟 -0.6，睡觉不下降     |
| 饱食度下降 | 空闲每分钟 -0.5，学习/工作每分钟 -0.6，睡觉每分钟 -0.2 |
| 点击互动 | 10 秒冷却，心情 +3                       |
| 学习 | 5 / 15 / 40 分钟，学识分别 +1 / +3.5 / +10 |
| 睡觉 | 5 / 15 / 40 分钟，心情分别 +5 / +18 / +50 |
| Buff | 高级食物可降低饱食下降速度，高级玩具可降低心情下降速度；Buff 时长为物品固定属性 |

### 初始状态

首次启动且没有已有存档时，宠物初始金币为 0，学识为 0，心情为 100，饱食度为 100。已有存档会继续使用存档中的数值。

### 心情倍率

| 心情 | 倍率 |
| --- | ---: |
| 80-100 | 1.5x |
| 50-79 | 1.0x |
| 20-49 | 0.7x |
| 0-19 | 0.3x |

## 工作列表

每个工种提供多个时长档位，长档位时薪略高于短档位。学识不足时也可下拉查看档位和收益，仅禁止开始工作。

| 工作 | 学识要求 | 时长 | 金币收益 | 时薪(G/分钟) |
| --- | ---: | ---: | ---: | ---: |
| 捡瓶子 | 0 | 5 / 15 分钟 | 20 / 65 | 4.00 / 4.33 |
| 发传单 | 20 | 5 / 15 / 30 分钟 | 40 / 128 / 270 | 8.00 / 8.53 / 9.00 |
| 外卖骑手 | 100 | 10 / 30 分钟 | 95 / 300 | 9.50 / 10.00 |
| 咖啡师 | 200 | 10 / 30 / 60 分钟 | 105 / 330 / 690 | 10.50 / 11.00 / 11.50 |
| 宠物主播 | 300 | 15 / 30 / 60 分钟 | 180 / 375 / 780 | 12.00 / 12.50 / 13.00 |
| 程序员宠物 | 400 | 15 / 30 / 60 分钟 | 203 / 420 / 900 | 13.53 / 14.00 / 15.00 |
| 神秘顾问 | 500 | 20 / 40 / 60 分钟 | 310 / 640 / 1000 | 15.50 / 16.00 / 16.67 |

学识超过 500（最高工种要求）后，每多 1 点学识，工作收益 +0.2%，上限 +100%（学识 1000 时满加成）。旧存档中的「咖啡店帮工」自动映射到「咖啡师」。

## 存档

开发运行时，存档会写入项目根目录下的 `saves/` 文件夹。程序会维护 `save.json` 和 `save_backup.json`，保存时使用临时文件写入后替换正式存档，降低写入中断造成损坏的风险。

## 界面说明

PetBuddy 使用 `assets/Original static image/` 中的静态图片展示不同宠物状态。主窗口会用 `QPainter` 叠加状态文字和任务进度。

多数弹窗界面复用 `ui/dialog_styles.py` 中的温馨宠物主题配色，包括交互选择弹窗、商店、状态面板、设置面板和右键菜单。状态面板打开期间会持续刷新，因此任务剩余时间和 Buff 倒计时会保持实时变化。
